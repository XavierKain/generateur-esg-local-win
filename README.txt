🔧 RÉSOLUTION DE PROBLÈMES
===========================

❌ "Le système ne trouve pas le fichier spécifié"
   → Vérifiez que le dossier "python311" existe dans le même répertoire

❌ Le navigateur ne s'ouvre pas
   → Ouvrez manuellement: http://localhost:8501

❌ Port déjà utilisé
   → Fermez les autres applications Streamlit ou redémarrez l'ordinateur
